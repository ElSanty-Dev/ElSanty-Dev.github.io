# ¿Qué es Phyton?

Python es un lenguaje de programación de alto nivel, interpretado y de propósito general, conocido por su sintaxis limpia y fácil de leer, lo que lo hace ideal para principiantes. Funciona ejecutando el código línea por línea a través de un intérprete, lo que permite pruebas interactivas, y gestiona la memoria automáticamente para simplificar la tarea del programador. Se usa en una amplia variedad de aplicaciones, incluyendo desarrollo web, ciencia de datos, inteligencia artificial y automatización de tareas. 

# Las Variables en Phyton
Las variables en Python funcionan como "contenedores" etiquetados para almacenar datos. Se crean en el momento en que se les asigna un valor usando el signo de igual (=) y no es necesario declarar el tipo de dato explícitamente, ya que Python lo deduce automáticamente (tipado dinámico). Puedes asignarles diferentes tipos de valores y reasignarlos con un nuevo tipo más adelante. 
 ¿Cómo funcionan?
Contenedores de datos: Piensa en una variable como una etiqueta que apunta a un objeto (un valor) almacenado en la memoria.
Creación simple: Para crear una variable, solo necesitas un nombre (la etiqueta) y asignarle un valor con el signo =. Por ejemplo, nombre_usuario = "Ana" crea una variable nombre_usuario que contiene el texto "Ana".
Tipado dinámico: Python detecta automáticamente el tipo de dato que se asigna. Por ejemplo, edad = 25 (un entero) y precio = 9.99 (un flotante) se almacenan como tipos diferentes sin que tengas que especificarlos.
Reasignación: Puedes cambiar el valor de una variable en cualquier momento, e incluso cambiar su tipo. Por ejemplo, si después escribes edad = "veinticinco", la variable ahora contendrá una cadena de texto.
Uso: Una vez creada, puedes usar el nombre de la variable en tu código, y Python lo reemplazará con su valor. La función print() puede mostrar su contenido.
